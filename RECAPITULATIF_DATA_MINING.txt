┌────────────────────────────────────────────────────────────────────────────┐
│ 🔒 ANONYMISATION AUTOMATIQUE                                              │
└────────────────────────────────────────────────────────────────────────────┘

Le système remplace automatiquement les noms de clients :

    "Total Sénégal"        →  "Secteur Pétrolier"
    "Shell Sénégal"        →  "Secteur Pétrolier"
    "SAR Sénégal"          →  "Secteur Pétrolier"
    "Mine d'Or"            →  "Secteur Minier"
    "Nestlé Sénégal"       →  "Secteur Industriel"
    "Hôpital Principal"    →  "Secteur Médical"
    
Pour ajouter un client :

    // Dans src/shared/services/dataAnalyticsService.ts
    const CLIENT_TO_SECTOR = {
      'nouveau_client': 'Secteur Approprié',
    };

┌────────────────────────────────────────────────────────────────────────────┐
│ 📈 INDICATEURS CALCULÉS                                                   │

TONNAGE TOTAL
─────────────

RÉPARTITION PAR TYPE
─────────────────────
Catégories : Hydrocarbures, Dangereux, Non-Dangereux, Médicaux, Autres
Calcul : (Tonnage catégorie / Tonnage total) × 100

TAUX DE VALORISATION
─────────────────────
Méthodes : Valorisation énergétique, Recyclage, Compostage, Régénération
Calcul : (Tonnage valorisé / Tonnage total) × 100

RÉPARTITION PAR SECTEUR
────────────────────────
Anonymisé automatiquement (pas de noms de clients)
Groupement par secteur d'activité
