AJOUTER DES FILTRES
───────────────────
Exemple : Filtrer par type de déchet

1. Ajouter state :
   const [selectedWasteType, setSelectedWasteType] = useState('all');

2. Ajouter dans filtres :
   <select onChange={(e) => setSelectedWasteType(e.target.value)}>
     <option value="all">Tous types</option>
     {metrics.wasteBreakdown.map(wb => (
       <option value={wb.category}>{wb.categoryLabel}</option>
     ))}
   </select>

3. Filtrer dans loadData() :
   if (selectedWasteType !== 'all') {
     extracted.wasteBreakdown = extracted.wasteBreakdown.filter(
       w => w.category === selectedWasteType
     );
   }


AJOUTER UN GRAPHIQUE
────────────────────
Exemple : Donut Chart

<ResponsiveContainer width="100%" height={300}>
  <PieChart>
    <Pie
      data={data}
      cx="50%"
      cy="50%"
      innerRadius={60}  // ← Crée le trou central
      outerRadius={100}
      dataKey="value"
    >
      {data.map((entry, index) => (
        <Cell key={index} fill={COLORS[index % COLORS.length]} />
      ))}
    </Pie>
    <Tooltip />
  </PieChart>
</ResponsiveContainer>

PROBLÈME : Page blanche
───────────────────────
Solution :
  1. Console F12 → Voir erreurs
  2. Vérifier données : quickStats()
  3. Créer données test : quickCreate5()


PROBLÈME : Graphiques ne s'affichent pas
─────────────────────────────────────────
Solution :
  1. Vérifier que Recharts est installé :
     npm list recharts
     
  2. Si non installé :
     npm install recharts
     
  3. Redémarrer serveur :
     npm run dev


PROBLÈME : Filtres ne fonctionnent pas
───────────────────────────────────────
Solution :
  1. Ouvrir filtres avec bouton en haut
  2. Vérifier que les données existent
  3. Console : Voir si loadData() est appelé
  4. Changer de période (Mois → Année)


PROBLÈME : Export infographie ne fonctionne pas
────────────────────────────────────────────────
Solution :
  1. Vérifier que WebExportModal existe
  2. Console : Voir erreurs
  3. Tester avec données différentes
