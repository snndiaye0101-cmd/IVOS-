01 — Qu'est-ce qu'IVOS ?

IVOS est une plateforme web professionnelle (SaaS B2B) conçue pour les entreprises de transport et de gestion de déchets en Afrique de l'Ouest. Elle centralise l'ensemble des opérations quotidiennes en un seul outil.

Problèmes résolus

  • Fin des fichiers Excel dispersés — Tout est centralisé dans une seule application
  • Traçabilité réglementaire — Bordereaux BSD numériques avec signatures horodatées
  • Visibilité temps réel — État de la flotte, missions, finances en un coup d'œil
  • Communication centralisée — Chat interne par département, alertes automatiques
  • Multi-sites — Gérer plusieurs pays/filiales depuis une seule interface
  • Conformité — Audit trail complet, isolation des données par filiale (RLS)
💡 Progressive Web App (PWA)
IVOS fonctionne dans le navigateur mais peut s'installer sur téléphone comme une application native. Pas besoin de télécharger depuis un app store.

03 — Rôles Utilisateurs

Chaque utilisateur a un rôle qui détermine ce qu'il peut voir et faire dans le système. Les données sont isolées par filiale/pays (multi-tenant).

Rôle Périmètre Accès Principal
Super Admin Global (toutes filiales) Configuration système, gestion utilisateurs, toutes données, paramètres de sécurité
Directeur Pays Sa filiale Dashboard complet, données financières sensibles, validation stratégique, exports
Dispatcher Son site Création et affectation des missions, coordination flotte, suivi en temps réel
Superviseur Son site Validation terrain, contrôle qualité, signatures BSD, suivi opérations
Chauffeur Ses missions Réception missions, pré-départ, signatures BSD, chat, signalement pannes
Client Ses documents Consultation de ses bordereaux et missions uniquement
🔐 Isolation Multi-Tenant
Un utilisateur au Sénégal ne peut jamais voir les données de la Côte d'Ivoire, et vice-versa. C'est garanti au niveau de la base de données par des politiques Row Level Security (RLS).

Workflow d'inscription

  1. L'utilisateur crée un compte → statut En attente
  2. Un administrateur examine la demande
  3. Il approuve → Approuvé (avec fonction et rôle assignés)
  4. Ou il refuse → Rejeté

📜 Registre BSD (Bordereaux de Suivi des Déchets)

⚠️ Document Réglementaire Obligatoire
Le BSD est le document légal qui accompagne tout transport de déchets. IVOS le digitalise entièrement avec signatures électroniques horodatées.
SectionContenu
A — ProducteurNom, adresse, contact, NINEA (n° identification nationale)
B — CaractéristiquesDescription du déchet, état physique (solide/liquide/boue), code catégorie, conditionnement (benne, citerne, fût…), poids/volume estimés. Si dangereux : n° ONU, classe de danger
C — TransporteurEntreprise, n° licence, immatriculation véhicule, nom et permis chauffeur, date collecte + signature de confirmation
D — DestinationNom installation, n° licence, date réception, poids/volume réels, statut d'acceptation
SignaturesProducteur, Transporteur, Destinataire, Superviseur — horodatées

Workflow du BSD

Brouillon → Validé → En transit → Réceptionné → Signé

Statuts exceptionnels : Rejeté Annulé (acceptation partielle possible)

Génération PDF automatique du bordereau complet avec toutes les signatures et données.
