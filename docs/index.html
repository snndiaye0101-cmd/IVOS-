🚀 Démarrage Rapide

# 1. Installation
npm install
npm install -g supabase
# 2. Configuration Supabase
supabase start
supabase db push --local
npm run supabase:types
# 3. Configuration environnement
cp .env.example .env.local
# Éditer .env.local avec vos credentials
# 4. Lancer l'application
npm run dev
# Ouvre http://localhost:3000

🎯 Fonctionnalités Principales

✅ Multi-Tenant & RBAC

  • Architecture multi-pays/filiales
  • 6 rôles utilisateur (Super Admin → Driver)
  • Row Level Security (RLS) activé

✅ Gestion de Flotte

  • CRUD Véhicules (maintenance, documents)
  • CRUD Chauffeurs (permis, certifications)
  • Suivi statuts temps réel

✅ Ordres de Mission

  • Workflow : Brouillon → Validé → En cours → Terminé → Clôturé
  • Association Véhicule + Chauffeur + Route
  • Suivi kilométrique

🌟 Bordereau de Suivi des Déchets (BSD) - PRIORITÉ

  • Section A: Producteur/Client
  • Section B: Caractérisation du Déchet
  • Section C: Collecteur/Transporteur
  • Section D: Installation de Destination
  • Signatures numériques multi-parties (4)
  • Génération PDF via webhook n8n

✅ Reporting & Analytics

  • Dashboard avec KPIs
  • Graphiques interactifs (Recharts)
  • Exports PDF/Excel
  • Notifications temps réel
