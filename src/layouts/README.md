# README — shared/layouts

Ce dossier regroupe les layouts partagés (structures de pages, wrappers, templates) utilisés dans l’ensemble du système IVOS.

## Conventions
- Les layouts sont documentés (JSDoc/commentaire en tête).
- Les tests unitaires sont à placer dans `__tests__` ou à côté du layout.
- Respecter les conventions UI/UX du projet (voir `/docs/CONVENTIONS_CODE.md`).

## Contact
Pour toute question ou amélioration, contacter l’équipe UI IVOS.
