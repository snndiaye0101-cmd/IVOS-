# README — shared/styles

Ce dossier contient les fichiers de styles globaux et partagés du système IVOS.

- **index.css** : Styles globaux, Tailwind, variables CSS, utilitaires.
- Autres fichiers de styles partagés à placer ici.

## Conventions

- Utiliser Tailwind CSS pour la majorité des styles.
- Les styles spécifiques à un composant doivent rester dans le dossier du composant.
- Respecter les conventions UI/UX du projet (voir `/docs/CONVENTIONS_CODE.md`).
