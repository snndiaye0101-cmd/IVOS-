# README — app

Ce dossier contient le point d’entrée principal de l’application IVOS et la configuration globale.

- **main.tsx** : Point d’entrée, initialisation React, providers globaux.
- **vite-env.d.ts** : Types d’environnement Vite.
- **jest-globals.d.ts** : Types globaux pour les tests Jest.
