# README — shared/components

Ce dossier regroupe les composants partagés entre plusieurs modules du système IVOS.

- **ui/** : Composants UI universels (voir README dédié).
- Autres composants partagés à placer ici.

## Conventions
- Les composants sont documentés (JSDoc/commentaire en tête).
- Les tests unitaires sont à placer dans `__tests__` ou à côté du composant.
- Respecter les conventions UI/UX du projet (voir `/docs/CONVENTIONS_CODE.md`).
