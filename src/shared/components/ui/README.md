# README — shared/components/ui

Ce dossier contient les composants UI réutilisables et universels du système IVOS :

- **UniversalLoader.tsx** : Loader universel pour affichage d’attente (chargement global ou partiel).
- **withLoader.tsx** : HOC pour gestion du suspense/lazy loading sur les composants.

## Conventions

- Les composants sont documentés (JSDoc/commentaire en tête).
- Les tests unitaires sont à placer dans `__tests__` ou à côté du composant.
- Respecter les conventions UI/UX du projet (voir `/docs/CONVENTIONS_CODE.md`).
