│ 🔧 API DU SERVICE CERTIFICAT                                              │
└────────────────────────────────────────────────────────────────────────────┘

FONCTIONS PRINCIPALES
─────────────────────

canGenerateCertificate(operationId: string)
→ Vérifie si un certificat peut être généré
→ Retourne : { canGenerate: boolean, reason?: string }

generateCertificateFromOperation(operationId: string, generatedBy: string)
→ Génère automatiquement un certificat depuis une opération
→ Retourne : Certificate | null

getCertificates()
→ Récupère tous les certificats
→ Retourne : Certificate[]

getCertificateById(id: string)
→ Récupère un certificat par ID
→ Retourne : Certificate | null

getCertificateByNumber(certificateNumber: string)
→ Récupère un certificat par numéro
→ Retourne : Certificate | null

verifyCertificate(verificationCode: string)
→ Vérifie l'authenticité d'un certificat
→ Retourne : CertificateVerificationResult

sendCertificate(params: CertificateSendParams)
→ Envoie un certificat par email (simulé)
→ Retourne : Promise<void>

markCertificateAsSent(certificateId: string)
→ Change le statut en "sent"

markCertificateAsVerified(certificateId: string)
→ Change le statut en "verified"

getCertificateStats()
→ Statistiques des certificats
→ Retourne : { total, generated, sent, verified, thisMonth, thisYear }


FONCTIONS PDF
─────────────

generateCertificatePDF(certificate: Certificate)
→ Génère le Blob PDF
→ Retourne : Promise<Blob>

downloadCertificatePDF(certificate: Certificate)
→ Télécharge le PDF
→ Retourne : Promise<void>

previewCertificatePDF(certificate: Certificate)
→ Prévisualise le PDF
→ Retourne : Promise<string> (URL Blob)

PROBLÈME : Impossible de générer un certificat
───────────────────────────────────────────────
Solution :
  1. Vérifier statut BSD : doit être "cloturee"
  2. Vérifier Section 9 validée : bsdData.validatedAt présent
  3. Vérifier facture existe et statut >= "emise"
  4. Console : canGenerateCertificate('operation_id')


PROBLÈME : PDF ne se télécharge pas
────────────────────────────────────
Solution :
  1. Vérifier qrcode installé : npm list qrcode
  2. Console : Voir erreurs JavaScript
  3. Tester avec un certificat différent
  4. Vérifier bloqueur de popup


PROBLÈME : QR Code ne fonctionne pas
─────────────────────────────────────
Solution :
  1. Vérifier URL générée : doit contenir /certificate/verify/
  2. Scanner avec application QR Code tierce
  3. Copier/coller code manuellement
  4. Vérifier code existe : verifyCertificate('CODE')


PROBLÈME : Page de vérification blanche
────────────────────────────────────────
Solution :
  1. Vérifier route dans App.tsx
  2. Console : Voir erreurs React
  3. Tester URL directe : /certificate/verify
  4. Vérifier certificateService importé
