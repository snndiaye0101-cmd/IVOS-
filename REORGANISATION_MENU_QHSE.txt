                          🚀 UTILISATION IMMÉDIATE
═══════════════════════════════════════════════════════════════════════════════

1. Lancer le serveur :
   npm run dev

2. Accéder à la nouvelle section :
   Menu → REPORTING & IMPACT → Reporting QHSE

3. Tester les 3 onglets :
   ┌─────────────────────────────────────────────────────┐
   │  [Tableau de Bord] [Archives QHSE] [Certificats]   │
   └─────────────────────────────────────────────────────┘

4. Tester les filtres dans Archives :
   • Rechercher un document
   • Filtrer par type (VGP/BSD/Certificat)
   • Sélectionner un client
   • Définir une plage de dates
   • Réinitialiser les filtres
